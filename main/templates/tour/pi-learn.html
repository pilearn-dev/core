Jeder kann Inhalte einbringen

Jeder Benutzerin und jeder Benutzer kann Kurse vorschlage, erarbeiten und anbieten.

In den Kursforen kann man über Kursinhalte diskutieren.

Deine Inhalte können von anderen bearbeitet werden.

Die Community kontrolliert die Inhalte

Alle Benutzer können die Inhalte auf {{ site_name }} bewerten und schädliche Inhalte melden oder zur Löschung vorschlagen.

So wird eine durchgängig hohe Qualität durchgesetzt.

Richtlinien dafür werden in einem globalen Forum entwickelt und dokumentiert.

Reputation – Vertrauen von der Community

Durch die Bewertung der Inhalte zeigt die Community, wie sehr sie dir vertraut. Daraus errechnet sich eine Punktzahl – deine Reputation

Du erhältst Reputation, wenn ...

+3

dein Beitrag einen Upvote erhält

+1

ein Benutzer deinem Kurs beitritt

+5

dein Kurs einen Upvote erhält

max.
+50

ein Pull Request angenommen wird

Dir wird Reputation abgezogen, wenn ...

–3

dein Beitrag einen Downvote erhält

–5

dein Kurs einen Downvote erhält

Mit steigender Reputation erhältst du Privilegien zur Moderation der Inhalte ...

1 Beiträge bewerten
100 Kurse bewerten
1 Überall kommentieren
1 Beiträge bearbeiten
100 Beiträge löschen
200 Kurse sperren

... und kannst so zusammen mit den Moderierenden zum Erfolg von {{ site_name }} beitragen.
