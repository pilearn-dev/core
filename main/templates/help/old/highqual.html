Hilfethemen / Kurse / Hochwertige Inhalte

Kurse, die besonders hochqualitative Inhalte haben, können von Moderatoren mit dem Badge (Abzeichen) hochwertige Inhalte ausgezeichnet werden. Dieses Abzeichen kann nur von Administratoren wieder entfernt werden.

Kurse mit hochwertigem Inhalt bringen maximal 50 Reputationspunkte und damit doppelt so viel wie normale Kurse.

Kurse mit hochwertigem Inhalt können nicht mit einem geringe inhaltliche Qualtiät-Flag gemeldet werden. Nur Moderatoren können einen Kurs mit hochwertigen Inhalten löschen.
